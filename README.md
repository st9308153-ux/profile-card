# Profile Card Project

Створити репозиторій. В репозиторій додати Readme.md, описати що це за проєкт, надати посилання на Figma.
Дотримуватись git flow при роботі над проєктом. Кожна окрема секція повинна робитись в новій гілці.
Верстка повинна відповідати макету з Figma (шрифти, кольори, відступи, розміри елементів).
Використовувати семантичну HTML-розмітку.
Дотримуватись чистого та структурованого CSS.
Забезпечити валідність HTML та CSS (без критичних помилок), перевірити валідність можна тут https://validator.w3.org/. 
Код має бути відформатований та зрозумілий (відступи, неймінг, структура файлів).
Кожен завершений етап роботи оформлювати через pull request.
Після завершення проєкту задеплоїти його на GitHub Pages та додати посилання в README.
Проєкт повинен коректно відображатися в сучасних браузерах (Chrome, Firefox, Edge).
Заборонено копіювати чужі рішення — проєкт має бути виконаний самостійно.

## Технології
- HTML5
- CSS3 (Flexbox)
